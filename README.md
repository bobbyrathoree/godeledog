# The Hole in Every Map

An interactive visual explanation of Gödel's incompleteness theorems — and a
love letter to Kurt Gödel.

The experience distinguishes the theorem's external proof from the undecidable
sentence inside a selected formal system, then uses Rosser's refinement to state
the consistency-only version precisely. A real Gödel-style number encoder, a
two-predicate diagonal mirror, a hand-grown MIU proof tree, and a document-height
biographical tree let the visitor touch the distinction between truth and proof.

The biography stays biography. The mathematics stays mathematics. They share a
tree, but neither is used to explain away the other.

## Run locally

```sh
python3 -m http.server 7480
```

Open <http://localhost:7480>.

No build step or dependencies are required.
